# JobTrackr 🚀

A Django-based job tracking system to help users manage job applications, track progress, and stay organized.

---

## 🌐 Live Demo

👉 https://your-railway-link.up.railway.app  
*(replace after deployment)*

---

## ⚙️ Tech Stack

- Python 3.11
- Django 5.x
- PostgreSQL (Railway)
- Gunicorn (Production server)
- WhiteNoise (Static files)
- Railway (Deployment)

---

## 📦 Features

- User authentication system
- Job application tracking
- Dashboard overview
- Status updates (Applied, Interview, Rejected, etc.)
- Admin panel for management
